# DevTraverse Missions

Mission codebases for DevTraverse — production-grade repositories used for skill evaluation. Each mission is a real codebase with a failing test suite. Public for GitHub Actions free runner minutes.

## Structure

```
missions/
└── nodejs-backend/
    └── add-rate-limiting/   # Implement rate limiting on an Express API
```

## How Missions Work

Each mission is a self-contained codebase with:
- A working application with a deliberate gap to fill
- A test suite that defines what "done" looks like
- No hidden gotchas — you can see exactly what you're being evaluated against
